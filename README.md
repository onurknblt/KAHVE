Bu, create-next-app ile oluşturulmuş bir Next.js projesidir.

Başlangıç
Öncelikle geliştirme sunucusunu (development server) çalıştırın:

Bash
npm run dev
# veya
yarn dev
# veya
pnpm dev
# veya
bun dev
Sonucu görmek için tarayıcınızda http://localhost:3000 adresini açın.

app/page.tsx dosyasını değiştirerek sayfayı düzenlemeye başlayabilirsiniz. Siz dosyayı düzenledikçe sayfa otomatik olarak güncellenir.
